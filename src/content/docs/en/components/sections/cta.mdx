## Theme Variants

| Theme | Component | Style |
|-------|-----------|-------|
| **Liquid** | `CTALiquid.astro` | Animated liquid background with gradient CTA |
| **Glass** | `CTAGlass.astro` | Frosted glass CTA card |
| **Neo** | `CTANeo.astro` | Bold, terminal-style CTA |
| **Luxury** | `CTALuxury.astro` | Gold-accented premium CTA |
| **Minimal** | `CTAMinimal.astro` | Clean, centered CTA |
| **Aurora** | `CTAAurora.astro` | Gradient mesh background CTA |

All variants are located at `src/components/sections/cta/`.

## Content Source

CTA data is loaded from `src/locales/{lang}/cta.json` via `useTranslations()`.

<Callout type="info" title="No Props">
  CTA components take no external props. To change CTA text or button labels, edit `src/locales/{lang}/cta.json`.
</Callout>

## Customizing

To customize:
1. Edit `src/components/sections/cta/CTA{Theme}.astro`.
2. Adjust styles in `src/styles/components/cta/`.
3. Change content in `src/locales/{lang}/cta.json`.
