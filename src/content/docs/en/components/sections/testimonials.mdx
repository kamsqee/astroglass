The Testimonials section displays social proof and customer reviews.

## Theme Variants

| Theme | Component | Style |
|-------|-----------|-------|
| **Liquid** | `TestimonialLiquid.astro` | Flowing testimonial cards |
| **Glass** | `TestimonialGlass.astro` | Frosted glass quote cards |
| **Neo** | `TestimonialNeo.astro` | Bold, high-contrast testimonials |
| **Luxury** | `TestimonialLuxury.astro` | Elegant gold-accent reviews |
| **Minimal** | `TestimonialMinimal.astro` | Clean, minimal quote layout |
| **Aurora** | `TestimonialAurora.astro` | Gradient background testimonials |

All variants are located at `src/components/sections/testimonial/`.

## Content Source

Testimonial data is loaded from `src/locales/{lang}/testimonial.json` via `useTranslations()`. The JSON typically contains reviewer names, titles, quotes, and avatar info.

<Callout type="info" title="No Props">
  Testimonial components take no external props. To change quote text, names, or structure, edit `src/locales/{lang}/testimonial.json`.
</Callout>

## Customizing

To customize:
1. Edit `src/components/sections/testimonial/Testimonial{Theme}.astro`.
2. Adjust styles in `src/styles/components/testimonial/`.
3. Change content in `src/locales/{lang}/testimonial.json`.
