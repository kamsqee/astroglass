AstroGlass is configured via files in `src/config/` and standard CSS custom properties.

## Configuration Files

<FileTree>
- src/
  - config/
    - **themes.ts** ‚Äî Theme definitions (name, colors, sections, premium flag)
    - **locales.ts** ‚Äî Supported languages and locale metadata
    - **navigation.ts** ‚Äî Main site navigation structure
    - **docs.ts** ‚Äî Documentation sidebar and version config
</FileTree>

## Styling with Tailwind CSS v4

This project uses **Tailwind CSS v4** via the `@tailwindcss/vite` plugin. There is no `tailwind.config.mjs` file ‚Äî all configuration is done through CSS.

Colors and theme variables are defined as CSS custom properties in `src/styles/global.css` and `src/styles/_themes.css`:

```css title="global.css"
:root {
  --p: 260 80% 60%;   /* Primary (HSL) */
  --s: 310 80% 60%;   /* Secondary */
  --a: 190 90% 60%;   /* Accent */
  --b1: 220 20% 10%;  /* Base background */
  --bc: 220 20% 90%;  /* Base content (text) */
}
```

<Callout type="tip" title="Changing Colors">
  To change the site's color scheme, update the HSL values in `src/styles/global.css`. Theme-specific overrides live in `src/styles/_themes.css` using `[data-theme="..."]` selectors.
</Callout>

## Theme Configuration

