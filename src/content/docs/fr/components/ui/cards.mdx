Les cartes sont l'élément constitutif fondamental du système de design en verre (glass design).

## Variantes

Nous prenons en charge plusieurs styles de cartes via la propriété (prop) `variant`.

| Variante | Description | Style Visuel |
| :--- | :--- | :--- |
| `glass` | Verre dépoli standard | Flou + Bordure |
| `solid` | Arrière-plan opaque | Contraste élevé |
| `outline` | Bordure uniquement | Minimaliste |

## Exemples en Direct

<div className="grid gap-6 md:grid-cols-2 lg:grid-cols-3 not-prose my-8">
  <Card variant="glass" className="p-6">
    <h3 className="text-lg font-bold mb-2 text-white">Carte en Verre (Glass)</h3>
    <p className="text-sm opacity-80">Effet verre dépoli standard avec un flou et une bordure subtile.</p>
  </Card>

  <Card variant="solid" className="p-6">
    <h3 className="text-lg font-bold mb-2 text-white">Carte Pleine (Solid)</h3>
    <p className="text-sm opacity-80">Arrière-plan opaque pour un contraste élevé.</p>
  </Card>

  <Card variant="outline" className="p-6">
    <h3 className="text-lg font-bold mb-2 text-white">Carte Contour (Outline)</h3>
    <p className="text-sm opacity-80">Style minimaliste uniquement avec une bordure.</p>
  </Card>
</div>

## Utilisation du Code

```tsx
<Card variant="glass" className="p-6">
  <h3>Carte en Verre (Glass Card)</h3>
  <p>Ceci crée un bel effet de profondeur.</p>
</Card>
```
