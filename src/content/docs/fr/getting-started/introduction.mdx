AstroGlass est un thème Astro premium conçu pour les startups et les créateurs qui souhaitent se démarquer. Il présente un système de conception **Glassmorphism** unique, construit sur Tailwind CSS v4.

## Pourquoi AstroGlass ?

- **Design Premium** : 6 variantes de thèmes complètes avec des effets de profondeur et de flou.
- **Performance Hybride** : Contenu généré statiquement avec des capacités côté serveur.
- **Expérience Développeur** : Collections de contenu typées et configuration facile.
- **Entièrement i18n** : Prise en charge intégrée de 7 langues, facilement extensible.

## Qu'est-ce qui est Inclus ?

Le modèle est livré avec 6 thèmes entièrement conçus, chacun proposant plus de 12 sections de page (Héros, À propos, Fonctionnalités, Tarification, Témoignages, FAQ, CTA, Contact, En-tête, Pied de page, plus de nouvelles sections pour la Landing Page), une page Portfolio dédiée, une page 404 personnalisée, ainsi qu'un système de blog et une documentation complète.
