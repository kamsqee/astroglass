Démarrer est facile. Clonez le dépôt et installez les dépendances.

## Prérequis

- Node.js v20.0.0 ou supérieur
- pnpm v8+ (recommandé) ou npm/yarn

## Démarrage Rapide

<Steps>
1. **Cloner le dépôt**
   
   ```bash frame="terminal"
   git clone https://github.com/kamsqee/astroglass.git my-project
   ```

2. **Entrer dans le répertoire**

   ```bash frame="terminal"
   cd my-project
   ```

3. **Installer les dépendances**

   ```bash frame="terminal"
   pnpm install
   ```

4. **Démarrer le serveur de développement**

   ```bash frame="terminal"
   pnpm dev
   ```

</Steps>

Visitez `http://localhost:4321` pour voir votre nouveau site.
