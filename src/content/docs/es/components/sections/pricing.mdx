La sección Precios (Pricing) muestra los planes y niveles de los productos. Cada tema tiene su propio tratamiento visual.

## Variantes de tema

| Tema | Componente | Estilo |
|-------|-----------|-------|
| **Liquid** | `PricingLiquid.astro` | Diseño de tarjetas fluidas con acentos degradados |
| **Glass** | `PricingGlass.astro` | Tarjetas de precios de vidrio esmerilado |
| **Neo** | `PricingNeo.astro` | Cuadrícula de precios de alto contraste |
| **Luxury** | `PricingLuxury.astro` | Precios con acentos dorados y sensación premium |
| **Minimal** | `PricingMinimal.astro` | Lista de precios limpia y sencilla |
| **Aurora** | `PricingAurora.astro` | Tarjetas de precios con fondo degradado |

Todas las variantes se encuentran en `src/components/sections/pricing/`.

## Fuente de contenido

Los datos de precios se cargan desde `src/locales/{lang}/pricing.json` a través de `useTranslations()`. El JSON contiene los nombres de los planes, descripciones, precios, características y el texto del CTA (llamado a la acción).

```json title="pricing.json (extracto)"
{
  "title": "Precios Simples",
  "subtitle": "Elige el plan que funcione para ti",
  "plans": {
    "starter": {
      "name": "Básico",
      "price": "19",
      "features": ["5 Proyectos", "Análisis Básico"]
    }
  }
}
```

<Callout type="info" title="Sin propiedades">
  Los componentes de precios no toman propiedades externas. Para cambiar los planes, precios o características, actualice `src/locales/{lang}/pricing.json`.
</Callout>

## Personalización

Para personalizar:
1. Edite `src/components/sections/pricing/Pricing{Theme}.astro`.
2. Ajuste los estilos en `src/styles/components/pricing/`.
3. Cambie el contenido en `src/locales/{lang}/pricing.json`.
