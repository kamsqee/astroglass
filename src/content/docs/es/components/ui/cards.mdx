Las tarjetas son el bloque de construcción fundamental del sistema de diseño de cristal (glass design).

## Variantes

Admitimos múltiples estilos de tarjetas a través de la propiedad (prop) `variant`.

| Variante | Descripción | Estilo Visual |
| :--- | :--- | :--- |
| `glass` | Cristal esmerilado estándar | Desenfoque + Borde |
| `solid` | Fondo opaco | Alto contraste |
| `outline` | Solo borde | Minimalista |

## Uso del Código

```tsx
<Card variant="glass" className="p-6">
  <h3>Tarjeta de Cristal (Glass Card)</h3>
  <p>Esto crea un hermoso efecto de profundidad.</p>
</Card>
```
