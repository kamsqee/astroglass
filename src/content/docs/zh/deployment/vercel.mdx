AstroGlass 您仅需进行最少量的配置，即可被部署到 **Vercel** 平台上。

## 前置要求

- 一个 [Vercel](https://vercel.com/) 账户。
- 本地已安装 **Node.js** v20+ 运行环境。
- 已安装 Vercel 命令行 CLI 工具 (`npm install -g vercel`)。

### 选项 2：通过命令行 CLI 进行部署

<Steps>
1. **登录 Vercel 系统去**
   
   ```bash title="Terminal" frame="terminal"
   npx vercel login
   ```

2. **立刻进行您的站之部署上云操作**
   
   ```bash title="Terminal" frame="terminal"
   pnpm run deploy:vercel
   ```
</Steps>
