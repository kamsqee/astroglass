# 404 页面

AstroGlass 内置了一个高级复杂的 404 错误页面，它可根据用户所选的主题全自动地去进行配置。

## 架构

位于 `src/pages/404.astro`。

该页面使用定义在 `src/styles/_themes.css` 中的 CSS 变量，渲染标准的 404 视觉效果（漂浮的粒子、发光的球体、玻璃卡片），并与当前激活的配色方案相匹配。

## 功能

- **主题自适应**: 如果用户在浏览“Luxury”主题时遇到 404，错误页面将是金色/黑色。如果在“Neo”主题，则是黑色/青色。
- **动态背景**: 具有动画元素，如漂浮的粒子和变形的斑点。
- **本地化**: 消息通过 `src/locales/{lang}/common.json` 进行完全翻译（键以 `notFound` 开头）。

## 使用

当用户请求不存在的路由时，Astro（以及 Cloudflare/Vercel 等大多数部署平台）会自动提供此页面。无需进行任何配置。
