卡片组件是构建起整套基于毛玻璃视觉设计规范（系统）的最为基础的基石模块部件。

## 组件变体 (Variants)

我们借由设定在 `variant` 参数属性来为您提供且全面支持多种迥异的卡片表现样式风格。

| 变体样式 | 描述说明 | 界面视觉风貌 |
| :--- | :--- | :--- |
| `glass` | 标准式磨砂半透明毛玻璃 | 模糊处理背景 + 外侧边框 |
| `solid` | 满色不透明纯色底背景 | 鲜明的强烈高对比度展现 |
| `outline` | 仅保留外轮廓式边框线 | 简约极简主义气息 |
