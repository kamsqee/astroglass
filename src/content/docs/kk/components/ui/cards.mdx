Карталар — шыны (glass) дизайн жүйесінің іргелі құрылыс блогы болып табылады.

## Нұсқалары (Variants)

Біз `variant` қасиеті (prop) арқылы карталардың бірнеше стильдерін қолдаймыз.

| Нұсқа | Сипаттамасы | Визуалды стилі |
| :--- | :--- | :--- |
| `glass` | Кәдімгі күңгірт шыны (frosted glass) | Бұлыңғырлық (Blur) + Жиек |
| `solid` | Мөлдір емес фон | Жоғары контраст |
| `outline` | Тек жиегі ғана | Минималистік |
