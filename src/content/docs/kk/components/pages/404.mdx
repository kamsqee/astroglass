# 404 беті

AstroGlass пайдаланушы таңдаған тақырыпқа автоматты түрде бейімделетін күрделі 404 Қате бетін қамтиды.

## Архитектура

`src/pages/404.astro` мекенжайында орналасқан.

Бұл бет `src/styles/_themes.css` ішінде анықталған CSS айнымалыларын пайдалана отырып, белсенді түстер палитрасына сәйкес келетін 404 визуалды элементтерін (қалқымалы бөлшектер, жарқыраған шарлар, шыны карталар) көрсетеді.

## Мүмкіндіктер

- **Тақырыпқа бейімделген**: Егер пайдаланушы "Luxury" тақырыбымен шолып, 404 қатесіне тап болса, қате беті Алтын/Қара болады. Егер "Neo" тақырыбында болса, Қара/Көгілдір болады.
- **Интерактивті фон**: Қалқымалы бөлшектер және пішінін өзгертетін дақтар сияқты анимациялық элементтерді қамтиды.
- **Локализация**: Хабарламалар `src/locales/{lang}/common.json` арқылы толық аударылған (`notFound` басталатын кілттер).

## Пайдалану

Бұл бетті пайдаланушы жоқ маршрутты сұраған кезде Astro (және Cloudflare/Vercel сияқты көптеген орналастыру платформалары) автоматты түрде көрсетеді. Ешқандай конфигурация қажет емес.
