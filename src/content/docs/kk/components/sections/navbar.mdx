Навигация жолағы (Header) компоненті әрбір тақырып бетінің жоғарғы жағында пайда болады. Әрбір тақырыптың әртүрлі навигация үлгілері, мобильді мәзір іске асырулары және визуалды стильдері бар өзінің бірегей тақырып дизайны бар.

## Тақырып нұсқалары

| Тақырып | Компонент | Стиль |
|-------|-----------|-------|
| **Liquid** | `HeaderLiquid.astro` | Бұлыңғыр фоны бар мөлдір тақырып |
| **Glass** | `HeaderGlass.astro` | Аязды әйнек навигация жолағы |
| **Neo** | `HeaderNeo.astro` | Қалың, жоғары контрастты тақырып |
| **Luxury** | `HeaderLuxury.astro` | Алтын екпіні бар премиум тақырып |
| **Minimal** | `HeaderMinimal.astro` | Таза, минималистік навигация |
| **Aurora** | `HeaderAurora.astro` | Градиентті екпіні бар тақырып |

Барлық нұсқалар `src/components/layout/header/` мекенжайында орналасқан.

## Мүмкіндіктер

Әрбір тақырып нұсқасы әдетте мыналарды қамтиды:
- **Жұмыс үстелі (Desktop) навигациясы**: Ашылмалы мәзірлері бар көлденең навигация
- **Мобильді навигация**: Сырғымалы тартпасы бар гамбургер мәзірі
- **Тақырып ауыстырғышы**: Біріктірілген тақырып таңдау компоненті
- **Тіл ауыстырғышы**: Жалауша белгішелері бар аймақ таңдау құралы
- **Логотип**: Бренд логотипі компоненті

## Навигация конфигурациясы

Навигация сілтемелері `src/config/navigation.ts` файлында конфигурацияланған (соның ішінде Luxury тақырыбының кеңейтілген навигациясы).

Навигация белгілері `src/locales/{lang}/nav.json` арқылы аударылады.

<Callout type="info" title="Props жоқ">
  Тақырып компоненттері сыртқы props қабылдамайды. Навигация элементтері конфигурация файлдарында анықталады және белгілер (labels) жергілікті JSON файлынан жүктеледі.
</Callout>
