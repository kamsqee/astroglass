Бағалау (Pricing) бөлімі өнім жоспарлары мен деңгейлерін көрсетеді. Әрбір тақырыптың өзіндік визуалды стилі бар.

## Тақырып нұсқалары

| Тақырып | Компонент | Стиль |
|-------|-----------|-------|
| **Liquid** | `PricingLiquid.astro` | Градиентті екпіндері бар сұйық карта макеті |
| **Glass** | `PricingGlass.astro` | Аязды әйнек баға карталары |
| **Neo** | `PricingNeo.astro` | Жоғары контрастты баға торы |
| **Luxury** | `PricingLuxury.astro` | Премиум сезімі бар алтын екпінді баға |
| **Minimal** | `PricingMinimal.astro` | Таза, қарапайым баға тізімі |
| **Aurora** | `PricingAurora.astro` | Градиентті фондық баға карталары |

Барлық нұсқалар `src/components/sections/pricing/` мекенжайында орналасқан.

## Мазмұн көзі

Баға деректері `useTranslations()` арқылы `src/locales/{lang}/pricing.json` файлынан жүктеледі. JSON құрамында жоспар атаулары, сипаттамалар, бағалар, мүмкіндіктер және CTA мәтіні бар.

```json title="pricing.json (үзінді)"
{
  "title": "Қарапайым баға",
  "subtitle": "Сізге сәйкес келетін жоспарды таңдаңыз",
  "plans": {
    "starter": {
      "name": "Бастапқы",
      "price": "19",
      "features": ["5 жоба", "Негізгі аналитика"]
    }
  }
}
```

<Callout type="info" title="Props жоқ">
  Баға компоненттері сыртқы props қабылдамайды. Жоспарларды, бағаларды немесе мүмкіндіктерді өзгерту үшін `src/locales/{lang}/pricing.json` файлын жаңартыңыз.
</Callout>

## Баптау

Баптау үшін:
1. `src/components/sections/pricing/Pricing{Theme}.astro` файлын өңдеңіз.
2. `src/styles/components/pricing/` ішіндегі стильдерді реттеңіз.
3. `src/locales/{lang}/pricing.json` файлындағы мазмұнды өзгертіңіз.
