Ерекшеліктер (Features) бөлімі өнімнің немесе қызметтің артықшылықтарын көрсетеді. Әрбір тақырып бірегей орналасуды қамтамасыз етеді - карта торларынан анимациялық тізімдерге дейін.

## Тақырып нұсқалары

| Тақырып | Компонент | Макет (Layout) |
|-------|-----------|--------|
| **Liquid** | `FeaturesLiquid.astro` | Сұйық меңзерді апару (hover) әсерлері бар анимациялық мүмкіндік карталары |
| **Glass** | `FeaturesGlass.astro` | Аязды әйнек мүмкіндіктері карталары |
| **Neo** | `FeaturesNeo.astro` | Өткір жиектері бар қалың тордың орналасуы |
| **Luxury** | `FeaturesLuxury.astro` | Алтын реңктері бар талғампаз тор |
| **Minimal** | `FeaturesMinimal.astro` | Таза, қарапайым тізім пішімі |
| **Aurora** | `FeaturesAurora.astro` | Карта торы бар градиенттік торлы фон |

Барлық нұсқалар `src/components/sections/features/` мекенжайында орналасқан.

## Мазмұн көзі

Мүмкіндік деректері `useTranslations()` арқылы `src/locales/{lang}/features.json` файлынан жүктеледі. JSON құрылымы әдетте мыналарды қамтиды:

```json title="features.json (үзінді)"
{
  "title": "Неліктен бізді таңдайсыз?",
  "subtitle": "Өнімділік пен пайдаланудың қарапайымдылығы үшін жасалған",
  "items": [
    {
      "title": "Найзағайдай жылдам",
      "description": "Жүктеу уақытын секундтан аз ету үшін статикалық генерация."
    }
  ]
}
```

<Callout type="info" title="Қосымша параметрлер (Props) жоқ">
  Features компоненттері сыртқы props қабылдамайды. Мүмкіндік элементтерін, тақырыптарды немесе сипаттамаларды өңдеу үшін `src/locales/{lang}/features.json` файлын жаңартыңыз.
</Callout>

## Баптау

Баптау үшін:
1. `src/components/sections/features/Features{Theme}.astro` файлындағы тақырып нұсқасын өңдеңіз.
2. `src/styles/components/features/` ішіндегі стильдерді реттеңіз.
3. `src/locales/{lang}/features.json` файлындағы мазмұнды өзгертіңіз.
