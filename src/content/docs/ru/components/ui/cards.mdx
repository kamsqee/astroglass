Карточки — это основной строительный блок дизайн-системы glass.

## Варианты

Мы поддерживаем несколько стилей карточек через пропс `variant`.

| Вариант | Описание | Визуальный стиль |
| :--- | :--- | :--- |
| `glass` | Стандартное матовое стекло | Размытие + Граница |
| `solid` | Непрозрачный фон | Высокий контраст |
| `outline` | Только граница | Минималистичный |
