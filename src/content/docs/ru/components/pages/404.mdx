# Страница 404

AstroGlass включает сложную страницу ошибки 404, которая автоматически адаптируется к активной теме пользователя.

## Архитектура

Расположена в `src/pages/404.astro`.

Страница использует CSS переменные, определенные в `src/styles/_themes.css`, для рендеринга визуальных эффектов (плавающие частицы, светящиеся сферы, стеклянные карточки), соответствующих активной цветовой палитре.

## Возможности

- **Адаптация темы**: Если пользователь использует тему "Luxury" и попадает на 404, страница будет Золотой/Черной. Если "Neo" — Черной/Голубой.
- **Интерактивный фон**: Включает анимированные элементы, такие как плавающие частицы и морфинг-фигуры.
- **Локализация**: Полностью переведенные сообщения через `src/locales/{lang}/common.json` (ключи, начинающиеся с `notFound`).

## Использование

Эта страница автоматически обслуживается Astro (и большинством платформ, таких как Cloudflare/Vercel), когда пользователь запрашивает несуществующий маршрут. Настройка не требуется.
