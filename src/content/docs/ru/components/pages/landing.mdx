# Секции Лендинга

Главная точка входа шаблона (`src/pages/index.astro`) использует несколько специализированных секций, предназначенных для демонстрации возможностей шаблона. Они расположены в `src/components/sections/landing/`.

## Полоса Статистики (Stats Strip)

`src/components/sections/landing/StatsStrip.astro`

Горизонтальная полоса, отображающая ключевые метрики вашего проекта (например, "6 Тем", "66+ Компонентов").

- **Особенности**: Анимированный счетчик чисел при прокрутке.
- **Конфигурация**: Редактируйте массив `stats` в frontmatter компонента или используйте `src/locales/{lang}/landing.json` для меток.

## Витрина Шаблона (Template Showcase)

`src/components/sections/landing/TemplateShowcase.astro`

Макет в стиле "Bento grid", обеспечивающий визуальную навигацию по различным частям шаблона (Темы, Компоненты, Блог, Документация и т.д.).

- **Особенности**: Эффекты при наведении, стеклянные карточки, адаптивная сетка.
- **Контент**: Управляется через `src/locales/{lang}/landing.json` в разделе `landing.showcase`.

## FAQ Главной (FAQ Home)

`src/components/sections/landing/FAQHome.astro`

Легкая секция FAQ в стиле аккордеона, разработанная для лендинга.

- **Особенности**: Плавные анимации раскрытия/сворачивания, доступные ARIA атрибуты.
- **Контент**: Вопросы и ответы хранятся в `src/locales/{lang}/landing.json` в разделе `landing.faq`.

## Как Это Работает (How It Works)

`src/components/sections/landing/HowItWorks.astro`

3-шаговая визуализация рабочего процесса (например, Клон -> Настройка -> Развертывание).

- **Особенности**: Пронумерованные шаги с иконками и градиентными фонами.
- **Контент**: Текст шагов в `src/locales/{lang}/landing.json`.
