Секция Testimonials отображает социальное доказательство и отзывы клиентов.

## Варианты тем

| Тема | Компонент | Стиль |
|------|-----------|-------|
| **Liquid** | `TestimonialLiquid.astro` | Плавные карточки отзывов |
| **Glass** | `TestimonialGlass.astro` | Матовые стеклянные карточки |
| **Neo** | `TestimonialNeo.astro` | Смелые, контрастные отзывы |
| **Luxury** | `TestimonialLuxury.astro` | Элегантные золотые обзоры |
| **Minimal** | `TestimonialMinimal.astro` | Чистая, минимальная компоновка |
| **Aurora** | `TestimonialAurora.astro` | Градиентный фон |

<Callout type="info" title="Без пропсов">
  Контент загружается из `src/locales/{lang}/testimonial.json`.
</Callout>
