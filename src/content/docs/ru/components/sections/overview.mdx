## Как это работает

Каждая секция имеет **6 тематических вариантов** (Liquid, Glass, Neo, Luxury, Minimal, Aurora). Страница `src/pages/[theme].astro` использует карту компонентов для отображения нужного варианта.

<Callout type="info" title="Без внешних пропсов">
  Секции не принимают пропсов. Весь контент загружается через `useTranslations()` из `src/locales/{lang}/{section}.json`.
</Callout>

## Структура файлов

```
src/components/sections/{section}/
├── {Section}Liquid.astro
├── {Section}Glass.astro
├── {Section}Neo.astro
├── {Section}Luxury.astro
├── {Section}Minimal.astro
└── {Section}Aurora.astro
```
