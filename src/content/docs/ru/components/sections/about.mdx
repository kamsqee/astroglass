Секция About представляет вашу компанию, продукт или историю.

## Варианты тем

| Тема | Компонент | Стиль |
|------|-----------|-------|
| **Liquid** | `AboutLiquid.astro` | Органические фоны |
| **Glass** | `AboutGlass.astro` | Матовые стеклянные панели |
| **Neo** | `AboutNeo.astro` | Смелая типографика, контраст |
| **Luxury** | `AboutLuxury.astro` | Элегантная компоновка |
| **Minimal** | `AboutMinimal.astro` | Чистый дизайн |
| **Aurora** | `AboutAurora.astro` | Градиентная сетка |

<Callout type="info" title="Без пропсов">
  Контент загружается из `src/locales/{lang}/about.json`.
</Callout>
