Секция Hero — первое, что видят посетители. Каждая тема имеет уникальный дизайн с кастомными анимациями.

## Варианты тем

| Тема | Компонент | Дизайн |
|------|-----------|--------|
| **Liquid** | `HeroLiquid.astro` | Анимированные SVG-блобы, градиентный текст |
| **Glass** | `HeroGlass.astro` | Матовые карточки с размытым фоном |
| **Neo** | `HeroNeo.astro` | Смелая типографика, высокий контраст |
| **Luxury** | `HeroLuxury.astro` | Золотые акценты, премиальная компоновка |
| **Minimal** | `HeroMinimal.astro` | Чистый, центрированный дизайн |
| **Aurora** | `HeroAurora.astro` | Градиентная сетка, геометрические элементы |

Все варианты находятся в `src/components/sections/hero/`.

## Как это работает

Каждый вариант Hero использует `useTranslations()` для загрузки ключей переводов: `hero.title`, `hero.subtitle`, `hero.cta`, `hero.liveBadge`.

<Callout type="info" title="Без пропсов">
  Hero-компоненты не принимают внешних пропсов. Для изменения текста редактируйте `src/locales/{lang}/hero.json`.
</Callout>
