Секция Footer отображается внизу каждой тематической страницы.

## Варианты тем

| Тема | Компонент | Стиль |
|------|-----------|-------|
| **Liquid** | `FooterLiquid.astro` | Градиентный подвал |
| **Glass** | `FooterGlass.astro` | Стеклянный подвал |
| **Neo** | `FooterNeo.astro` | Контрастный подвал |
| **Luxury** | `FooterLuxury.astro` | Премиальный подвал |
| **Minimal** | `FooterMinimal.astro` | Простой подвал |
| **Aurora** | `FooterAurora.astro` | Градиентный подвал |

<Callout type="info" title="Без пропсов">
  Контент загружается из `src/locales/{lang}/footer.json`.
</Callout>
