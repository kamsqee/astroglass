Секция CTA (Call to Action) побуждает посетителей к следующему шагу — регистрации, демо или дальнейшему исследованию.

## Варианты тем

| Тема | Компонент | Стиль |
|------|-----------|-------|
| **Liquid** | `CTALiquid.astro` | Анимированный жидкий фон с градиентным CTA |
| **Glass** | `CTAGlass.astro` | Матовая стеклянная CTA-карточка |
| **Neo** | `CTANeo.astro` | Смелый, терминальный стиль |
| **Luxury** | `CTALuxury.astro` | Золотые акценты |
| **Minimal** | `CTAMinimal.astro` | Чистый, центрированный CTA |
| **Aurora** | `CTAAurora.astro` | Градиентный фон |

<Callout type="info" title="Без пропсов">
  Контент загружается из `src/locales/{lang}/cta.json`.
</Callout>
