Секция Portfolio демонстрирует проекты, кейсы или примеры работ.

## Варианты тем

| Тема | Компонент | Стиль |
|------|-----------|-------|
| **Liquid** | `PortfolioLiquid.astro` | Плавная сетка с hover-анимациями |
| **Glass** | `PortfolioGlass.astro` | Стеклянные карточки проектов |
| **Neo** | `PortfolioNeo.astro` | Смелая masonry-компоновка |
| **Luxury** | `PortfolioLuxury.astro` | Элегантная презентация |
| **Minimal** | `PortfolioMinimal.astro` | Простая сетка |
| **Aurora** | `PortfolioAurora.astro` | Градиентная галерея |

<Callout type="info" title="Без пропсов">
  Контент загружается из `src/locales/{lang}/portfolio.json`.
</Callout>
