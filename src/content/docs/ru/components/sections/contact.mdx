Секция Contact предоставляет валидированные контактные формы.

## Варианты тем

| Тема | Компонент | Стиль |
|------|-----------|-------|
| **Liquid** | `ContactLiquid.astro` | Плавный дизайн формы |
| **Glass** | `ContactGlass.astro` | Стеклянная форма |
| **Neo** | `ContactNeo.astro` | Контрастная форма |
| **Luxury** | `ContactLuxury.astro` | Премиальная форма |
| **Minimal** | `ContactMinimal.astro` | Простая форма |
| **Aurora** | `ContactAurora.astro` | Градиентный фон |

## Валидация

Формы используют **Valibot** для типобезопасной валидации. Подробнее: [Руководство по формам](/docs/en/guides/forms-and-validation).

<Callout type="info" title="Без пропсов">
  Контент загружается из `src/locales/{lang}/contact.json`.
</Callout>
