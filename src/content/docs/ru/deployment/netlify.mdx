AstroGlass можно развернуть на **Netlify** с минимальной конфигурацией.

## Предварительные требования

- Аккаунт [Netlify](https://www.netlify.com/).
- **Node.js** v20+ установлен локально.

## Способы развёртывания

### GitHub (рекомендуется)

<Steps>
1. **Загрузите код на GitHub**.
2. **Импортируйте в Netlify**: [app.netlify.com](https://app.netlify.com/) > Add new site.
3. **Настройте сборку**: Build Command: `pnpm run build`, Publish Directory: `dist`.
4. **Нажмите Deploy site**.
</Steps>

### CLI

```bash frame="terminal"
pnpm run deploy:netlify
```

<Callout type="info" title="Поисковый индекс">
  Индекс генерируется автоматически во время `pnpm run build`.
</Callout>
