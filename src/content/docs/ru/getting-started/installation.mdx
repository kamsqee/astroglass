Начать работу просто. Клонируйте репозиторий и установите зависимости.

## Предварительные требования

- Node.js v20.0.0 или выше
- pnpm v8+ (рекомендуется) или npm/yarn

## Быстрый старт

<Steps>
1. **Клонируйте репозиторий**
   
   ```bash frame="terminal"
   git clone https://github.com/kamsqee/astroglass.git my-project
   ```

2. **Перейдите в директорию**

   ```bash frame="terminal"
   cd my-project
   ```

3. **Установите зависимости**

   ```bash frame="terminal"
   pnpm install
   ```

4. **Запустите сервер разработки**

   ```bash frame="terminal"
   pnpm dev
   ```

</Steps>

Откройте `http://localhost:4321`, чтобы увидеть ваш новый сайт.
