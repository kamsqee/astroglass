AstroGlass は、際立つことを目指すスタートアップやクリエイター向けに設計されたプレミアムな Astro テーマです。Tailwind CSS v4 を基盤として構築されたユニークな **グラスモーフィズム** デザインシステムを備えています。

## なぜ AstroGlass なのか？

- **プレミアムデザイン**: 奥行きとぼかし効果を持つ6つの完全なテーマバリエーション。
- **ハイブリッドパフォーマンス**: サーバーサイド機能を備えた静的生成コンテンツ。
- **開発者体験 (DX)**: 型安全なコンテンツコレクションと簡単な構成。
- **完全な i18n**: 7つの言語の組み込みサポートにより、容易に拡張可能。

## 含まれるもの

このテンプレートには6つの完全にデザインされたテーマが付属しており、それぞれに12以上のページセクション（Hero、About、Features、Pricing、Testimonials、FAQ、CTA、Contact、Header、Footer に加え、新しいランディングページセクション）、専用の Portfolio ページ、カスタム404ページに加え、ブログシステムと完全なドキュメントが提供されます。
