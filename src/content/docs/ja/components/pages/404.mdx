# 404 ページ

AstroGlass には、ユーザーが選択したテーマに自動的に適応する洗練された 404 エラーページが含まれています。

## アーキテクチャ

`src/pages/404.astro` に配置されています。

このページは `src/styles/_themes.css` で定義された CSS 変数を使用して、アクティブなカラーパレットに一致する標準的な 404 の視覚効果 (浮遊するパーティクル、光るオーブ、ガラスカードなど) をレンダリングします。

## 機能

- **テーマ対応**: ユーザーが「Luxury (ラグジュアリー)」テーマで閲覧中に 404 に遭遇した場合、エラーページはゴールド/ブラックになります。「Neo (ネオ)」であれば、ブラック/シアンになります。
- **インタラクティブな背景**: 浮遊するパーティクルや変形するブロブといったアニメーション要素が特徴です。
- **ローカリゼーション**: `src/locales/{lang}/common.json` (`notFound` から始まるキー) を介してメッセージが完全に翻訳されます。

## 使用法

ユーザーが存在しないルート (URL) を要求した際に、Astro (および Cloudflare/Vercel のようなほとんどのデプロイ プラットフォーム) によって自動的に提供されます。設定は必要ありません。
