AstroGlass は最小限の構成で **Netlify** にデプロイできます。

## 前提条件

- [Netlify](https://www.netlify.com/) アカウント。
- **Node.js** v20 以降がローカルにインストールされていること。
- Netlify CLI がインストールされていること (`npm install -g netlify-cli`)。

## デプロイ方法

### オプション 1: GitHub 統合 (推奨)

<Steps>
1. **GitHub にコードをプッシュする**
   
   プロジェクトが GitHub リポジトリにあることを確認してください。

2. **Netlify にインポート**
   
   [app.netlify.com](https://app.netlify.com/) に移動し、Add new site (新しいサイトを追加) > Import an existing project (既存のプロジェクトをインポート) の順に進みます。

3. **ビルド設定の構成**
   
   以下の設定を使用します：
   - **ビルド コマンド (Build Command)**: `pnpm run build`
   - **公開ディレクトリ (Publish Directory)**: `dist`

4. **デプロイ**
   
   「Deploy site」をクリックし、ビルドが完了するのを待ちます。
</Steps>

### オプション 2: CLI によるデプロイ

<Steps>
1. **Netlify にログイン**
   
   ```bash title="ターミナル (Terminal)" frame="terminal"
   npx netlify login
   ```

2. **デプロイ**
   
   ```bash title="ターミナル (Terminal)" frame="terminal"
   pnpm run deploy:netlify
   ```
</Steps>

<Callout type="info" title="検索インデックス">
  ビルド中 (`pnpm run build`) に検索インデックスが自動的に生成されます。デプロイされたサイトで検索機能が動作するためには、ビルドのステップが正常に完了していることを確認してください。
</Callout>
