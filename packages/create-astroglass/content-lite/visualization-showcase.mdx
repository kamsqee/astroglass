## Code Snippet

Here's an example of rendering a chart component:

```typescript
// Example: Chart configuration
const chartConfig = {
  type: 'line',
  data: {
    labels: ['Jan', 'Feb', 'Mar', 'Apr', 'May'],
    datasets: [{
      label: 'Revenue',
      data: [1200, 1900, 3000, 5000, 4200],
      borderColor: 'hsl(var(--p))',
      tension: 0.4,
    }],
  },
};
```
